Contacts Data

Chat requires you to provide a contact list data source. This will be a list of the users with access to chat.

Security screen

In what screen is your security component located?

Chat requires some form of validation to be able to use it. Use one of Fliplet's security components (e.g.: Email Validation, Login).
If a user accesses the Chat screen without being validated, the user will be redirected to the Security screen.

Contacts Directory

In what screen is your contact directory located?

This is required to link the Chat screen to directory and when a user taps on someone's profile the user will be redirected to the directory screen and the person's profile will open.
